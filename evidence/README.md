# Evidence Collection

This folder contains redacted evidence collected during the phishing email investigation.

## Evidence Files

| Evidence | Purpose |
|---|---|
| email-header-redacted.png | Shows sender information, reply-to details, subject, and email metadata |
| email-body-redacted.png | Shows phishing content and social engineering techniques |
| raw-email-header.png | Shows technical email header information used during analysis |
| authentication-results.png | Shows SPF, DKIM, and DMARC authentication results |
| phishing-investigation-workflow.png | Shows the investigation methodology followed |

## Privacy Notice

All personal information has been removed or anonymized. The evidence is presented for educational and defensive cybersecurity analysis.
